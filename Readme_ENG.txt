				  speedmapping sessions. This is a completely playable 
				  map, yet it's not ideologically fully completed and 
				  there may be some flaws. Feedback is especially desirable 
				  as further improvements will be made. Resources need 
				  to be included in a strict order:
				  -file PSXD.pke PSXD.wad PSX_TEX.pke vplusclr.wad
